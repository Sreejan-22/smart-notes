# smart-notes

A note taking application

Tech Stack

- React.js
- Express.js
- Mongoose + MongoDB

Features

- Create, update, delete notes
- Signup, Login

Backend part of the project can be found [here](https://github.com/Sreejan-22/smart-notes-backend)
